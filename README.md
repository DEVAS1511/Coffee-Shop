## 📖 About

**Haadhya Coffee** is a modern Coffee Shop web application built using **React.js**.

Users can:

- ☕ Browse premium coffee products
- 🛒 Add products to cart
- 💰 View product prices
- 📱 Enjoy a fully responsive design
